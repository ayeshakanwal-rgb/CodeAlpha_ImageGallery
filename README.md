## CodeAlpha - Image Gallery
This is my Image Gallery project for the CodeAlpha Internship (Frontend Development domain).

## Description
A simple responsive image gallery built using HTML, CSS, and JavaScript.
It displays a collection of images in a grid layout with hover effects.

## Features
- Responsive grid layout
- Hover zoom effect on images
- Click to open lightbox
- Next/Prev buttons for navigation
- Image category filters (Nature, Animals, food, etc.)

### Tech Stack
- HTML
- CSS
- JavaScript

### How to Run

1. Clone this repository
   git clone https://github.com/ayeshakanwal-rgb/CodeAlpha_ImageGallery.git

2. Open index.html in your browser:
  - Hover over images to see zoom effect.
  - Click on an image to open the lightbox.
  - Use Next / Prev buttons to navigate between images.
  - Use filter buttons to view specific categories.
